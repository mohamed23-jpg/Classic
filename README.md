# Codenames Classic - لعبة الجواسيس العربية

## نظرة عامة
لعبة كلمات جماعية متعددة اللاعبين بتصميم Neo-Tokyo Cyberpunk وواجهة عربية RTL كاملة.

**الخادم الخلفي:** `https://uuuuu-rup4.onrender.com/api`  
**تقنية:** HTML5 + CSS3 + JavaScript (بدون React)

---

## التغييرات الجديدة

### 1. الشعار (logo.png)
- تم تغيير الشعار من SVG نصي إلى `<img src="/assets/images/logo.png">`
- **يجب إضافة الملف:** `public/assets/images/logo.png`
- يظهر بشكل كبير ومتوهج في شاشة التسجيل (max-width: 300px)
- يظهر صغيراً في رأس القائمة
- إذا لم توجد الصورة يظهر نص احتياطي "CODENAMES"

### 2. الأفاتارات (8 شخصيات فقط)
تم إزالة: `dog, cat, dino, fox, lion, bear`  
الباقي:
| الاسم | الكود |
|-------|-------|
| جاسوس | `spy` |
| محقق | `detective` |
| مدير | `manager` |
| سامورائي | `samurai` |
| مافيا | `mafia` |
| جاسوسة | `spy_f` |
| نينجا | `ninja` |
| فضائي | `alien` |

كل أفاتار رسم SVG حقيقي بالكامل (ليس emoji). يتم حقنها تلقائياً بـ `Effects.populateAvatarGrid()`.

### 3. واجهة حية Neo-Tokyo (CSS جديد)
- **شاشة التسجيل:** خلفية ثلاثية التدرج الراديالي + خطوط مسح (scanlines) متحركة
- **بطاقة التسجيل:** زجاج (glassmorphism) + حد نيون يتنفس بألوان 3 (أحمر/أزرق/بنفسجي)
- **الشعار:** توهج neon ثلاثي اللون يتناوب بأنيميشن
- **البطاقات الرئيسية:** hover مع رفع + glow ملون حسب نوع البطاقة
- **اللوحات الجانبية:** كل لوحة لها لون مميز في الرأس
- **خلفية شبكة:** grid متحرك طفيف على الجسم كله
- **أنيميشن دخول:** كل بطاقة رئيسية تظهر من الأسفل بتأخير تدريجي

### 4. شجرة المستويات 1-100
**الملف:** `public/js/levels.js`

**الفتح:**
- زر "btn-levels-right" في الشريط الجانبي → لوحة "panel-levels"
- تُعرض معلومات اللاعب الحالية (مستواه + XP الحالي / التالي)

**بنية الشجرة:**
- صفوف متعرجة، 4 مستويات في الصف
- الصف الفردي: يسار ← يمين
- الصف الزوجي: يمين ← يسار (zigzag)
- لكل مستوى: دائرة + رقم + نقطة ذهبية إذا له مكافأة

**حالات المستويات:**
- 🔵 **مفتوح:** الأزرق النيوني
- ✨ **الحالي:** ذهبي متوهج يتنبض
- ⬛ **مقفل:** رمادي شفاف

**معادلة XP:** `xp_needed(n) = floor(80 × n^1.6)`

**المكافآت الكبرى:**
| المستوى | المكافأة |
|---------|---------|
| 8 | 🔓 فتح السوق |
| 10 | لقب "متعلم" + 1000 عملة |
| 20 | لقب "محترف" + 2000 عملة |
| 30 | لقب "خبير" + 4000 عملة |
| 50 | لقب "بطل" + 10000 عملة |
| 70 | لقب "سيد" + 15000 عملة |
| 90 | لقب "أسطورة حية" + 25000 عملة |
| 100 | لقب "مختم اللعبة" + 50000 عملة |

**الضغط على أي مستوى:** يفتح popup يعرض: رقم المستوى، XP المطلوب، الإجمالي، الحالة، المكافآت.
